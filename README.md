# Window_Guard
OpenVINO黑客松比赛项目
英文名称：Window_Guard
中文名称：窗口卫士

## 硬件
1. 树莓派4B 8G版本
2. 一个磁石开关
3. USB摄像头(MP4视频文件也可以)

## 软件(库)
1. OpenVINO
2. RPi

## 使用方法
本项目使用的OPenVINO是是**2021.3**版本，并使用了ARM插件，因此
没有使用神经计算棒。
相关环境搭建视频，请参考我的B站

- 树莓派添加声音：https://www.bilibili.com/video/BV1FU4y1j7iB/
